# On Web UI - Servers

#### Build Server

```bash
$ node ./servers/build_server.js               # Runs on port 5000
```

#### Handbook Code Server

```bash
$ node ./servers/handbook_code_server.js       # Runs on port 7000
```

#### OnRack API Proxy Server

```bash
$ node ./servers/onrack_api_proxy_server.js    # Runs on port 2000
```

#### Graph Stream Server

```bash
$ node ./servers/graph_stream_server.js        # Runs on port 8888
```
